编译型语言：执行之前，有专门的编译过程，通过编译系统将高级语言翻译成机器语言
解释型语言：在运行的时候才翻译，每个语句都是执行的时候才翻译。
pycache文件夹：
Python解释器将源码转换为字节码，然后再由解释器来执行这些字节码。
Python运行时先检查【本地是否有上述字节码文件】和【该字节码文件的修改时间是否在其源文件之后】，是就直接执行，否则重复上述步骤。这样就不需要转字节码了。
