python__init__和__new__

init和new对比

  1. init方法为初始化方法, new方法才是真正的构造函数。new方法必须返回值，init方法不需要返回值
  2. new方法默认返回实例对象供init方法，实例方法使用。
  3. init方法为初始化方法，为类的实例提供属性或完成动作。
  4. new方法可以用在下面二种情况：
    a.继承不可变数据类型时需要用到new方法
    b.用在元类，定制创建类对象

    Python 类 Property Staticmethod Classmethod

    classmethod 用于构造函数
    staticmethod 静态方法,可以看成一个独立的方法，只不过调用的时候得通过这个类
    property 把一个方法变成一个静态属性

    元类

    元类（英语：metaclass）是一种实例是类的类。普通的类定义的是特定对象的行为，元类定义的则是特定的类及其对象的行为
